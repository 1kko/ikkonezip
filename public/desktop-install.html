이코네Zip 데스크톱 앱

맥에서 한글 파일명을 정규화하고 ZIP으로 압축하는 도구. 오프라인 지원, 추적 없음.

최신 버전 다운로드 (DMG)

설치 방법

1DMG 파일 다운로드

위의 다운로드 버튼을 누르고 GitHub Releases 페이지에서 .dmg 파일을 받으세요.

2DMG 열고 Applications 폴더로 드래그

다운로드 받은 DMG 파일을 더블클릭한 뒤, 이코네Zip 아이콘을 Applications 폴더로 끌어다 놓으세요.

3처음 실행 시: 우클릭 → 열기

Applications 폴더의 이코네Zip 아이콘을 우클릭(또는 Control+클릭) → 열기를 선택하세요. 보안 경고가 뜨면 "열기"를 다시 누르세요.

이후부터는 일반 더블클릭으로 실행됩니다.

왜 보안 경고가 뜨나요?

이 앱은 Apple Developer Program에 등록되어 있지 않습니다. macOS의 Gatekeeper는 Apple이 검증하지 않은 앱에 대해 경고를 표시합니다.

코드는 GitHub에서 공개되어 있으니 직접 확인하실 수 있습니다. 우려되시면 PWA 버전을 웹사이트에서 바로 사용하셔도 동일한 기능을 제공합니다.
